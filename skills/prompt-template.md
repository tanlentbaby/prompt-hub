模板生成器 - 根据需求自动生成提示词模板

## 用途

当你需要一个新的提示词模板，但不知道如何开始时，使用此命令。
告诉我你的需求，我会：
1. 分析需求
2. 设计模板结构
3. 生成模板内容
4. 提供变量定义
5. 准备 Skills 配置

---

## 输入

**需求描述**: {{ requirements }}

例如：
- "我需要一个代码审查模板"
- "帮我创建一个数据分析报告模板"
- "想要一个生成 SQL 存储过程的模板"

---

## 处理流程

### 1. 需求分析

分析你的需求，确定：
- 模板类型（代码/文档/配置等）
- 目标用户（开发者/产品经理/运维等）
- 输出格式（Markdown/YAML/JSON 等）
- 复杂度（简单/标准/复杂）

### 2. 模板设计

设计模板结构：
- 角色设定
- 任务描述
- 输入参数
- 处理要求
- 输出格式

### 3. 变量提取

识别模板中需要的变量：
- 必需参数（无默认值）
- 可选参数（有默认值）

### 4. 示例生成

提供使用示例：
- 示例输入
- 预期输出


提供需要添加到 `skills/prompt-show.md` 和 `skills/prompt-fill.md` 的代码。

