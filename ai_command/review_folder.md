# AI 命令：审查文件夹

**用途**: 审查文件夹的代码质量、结构和一致性

## 模板

请审查以下文件夹：

- **文件夹名称**: {folderName}
- **文件夹路径**: {folderPath}
- **审查类型**: {reviewType:-代码质量/架构/安全}
- **选中文件列表**: {selectedFiles}

## 审查要求

请进行以下方面的审查：

1. **代码质量审查**
   - 代码规范性检查
   - 命名规范检查
   - 注释和文档完整性
   - 代码复杂度分析

2. **架构审查**
   - 模块划分合理性
   - 依赖关系检查
   - 接口设计评估
   - 设计模式使用评估

3. **安全审查**
   - 敏感信息泄露检查
   - 输入验证检查
   - 权限控制检查
   - 依赖安全检查

4. **一致性审查**
   - 代码风格一致性
   - 架构模式一致性
   - 命名约定一致性
